Burger and Breakfast

B&B#

Ingrédients#

Pour 2 personnes :

  • 2 muffins anglais;
  • beurre salé ;
  • 2 œufs ;
  • 4 tranches de fromage a burger ;
  • 4 tranches de bacon.

Accessoires#

2 Cercles de cuisines

Préparation#

  1. Faire griller les muffins au grille pain, puis les beurrer légèrement ;
  2. Faire chauffer la poêle puis mettre les noisettes de beurre dans les cercles sur la poêle chaude ;
  3. Mettre immédiatement les œufs dans les cercles puis casser le jaune avec un peu de poivre ;
  4. Placer le fromage sur les muffins ;
  5. Mettre les œufs cuit sur le muffins avec le fromage et cuire le bacon pour le mettre sur les œufs ;
  6. Refermer le Burger et déguster chaud.
